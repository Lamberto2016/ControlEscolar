Documentos de requisitos v1.0

Requisitos funcionales:
RF001: El sistema debe permitir el registro de alumnos
RF002: El sistema debe imprimir reporte de calificaciones
RF003: El sistema debe permitir la carga de materias.

Requisitos no funcionales:
RNF001: El sistema debe ser usable.
RNF002: El sistema debe ser rapido.
RNF003: El sistema debe ser seguro.

Listas de caso de uso:
CU001: Registrar alumno.
CU002: Imprimir cardex.
CU003: Cargar materias

Concluciones:
Este sistema sirve para gestionar los servicios del departamento de control escolar.
